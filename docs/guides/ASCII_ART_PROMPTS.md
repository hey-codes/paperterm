# ASCII & Terminal Art Prompt Templates

Generate terminal-inspired, ASCII-aesthetic artwork optimized for e-ink displays using AI image generators.

## Design Philosophy

Terminal/ASCII art works exceptionally well on e-ink because:
- **Monochrome by nature** - Designed for single-color displays
- **High contrast** - Text on dark/light backgrounds
- **Grid-based** - Aligns with pixel structure
- **Nostalgic aesthetic** - Retro computing vibes
- **Clean edges** - No gradients needed

## Post-Processing for E-Ink

After generating, process the image:

1. **Convert to grayscale** - Remove any color
2. **Increase contrast** - Make blacks blacker, whites whiter
3. **Resize to 1236 × 1648** - Kindle PW5 dimensions
4. **Apply dithering** - Floyd-Steinberg for any gradients
5. **Test on device** - Verify readability
