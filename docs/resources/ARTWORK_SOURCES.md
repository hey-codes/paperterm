## Art Styles That Work Well

### High Contrast Styles
1. **Woodcut / Linocut** - Bold black shapes on white
2. **Silhouettes** - Pure black on white
3. **Stencil Art** - Clean edges, limited tones
4. **High-key Photography** - Bright, minimal shadows

### Texture-Based Styles
1. **Stippling / Pointillism** - Natural dithering
2. **Crosshatching** - Classic illustration technique
3. **Halftone** - Print-inspired dots
4. **Engraving** - Fine line detail

### Minimalist Styles
1. **Line Art** - Single continuous lines
2. **Geometric** - Shapes and patterns
3. **Typography** - Text as art
4. **Negative Space** - Strategic emptiness

### Asian Influences
1. **Sumi-e** - Japanese ink wash
2. **Zen Circles (Ensō)** - Meditative brushwork
3. **Chinese Calligraphy** - Elegant strokes
4. **Ukiyo-e** - Japanese woodblock prints

## Quality Checklist

Before adding artwork:

- [ ] Resolution: 1236 × 1648 pixels (or can be scaled)
- [ ] Works in grayscale (not color-dependent)
- [ ] High contrast (clear blacks and whites)
- [ ] No fine text that would become illegible
- [ ] Appropriate license for personal use
- [ ] Tested with dithering preview
