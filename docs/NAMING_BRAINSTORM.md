# Repository Naming Brainstorm

Documented for future reference. Final choice: **paperterm**

## Selected Name

### `paperterm`
- **Meaning**: Paper + Terminal
- **Why chosen**: Captures the e-ink "paper" display + terminal/ASCII aesthetic
- **Vibe**: Technical, clean, descriptive

## Naming Principles

Good repo names should be:
- **Memorable** - Easy to recall
- **Typeable** - No special characters, short
- **Meaningful** - Hints at purpose
- **Unique** - Not already taken
- **Personal** - Reflects your style
